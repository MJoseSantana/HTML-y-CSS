Modelo de Objetos del Documento (DOM)

El DOM es una interfaz de programación para documentos HTML y XML. Representa la estructura del documento como un árbol de objetos que se puede manipular con lenguajes de programación como JavaScript.

Por ejemplo, considera el siguiente HTML:

            
              <html>
                <head>
                  <title>Mi Página</title>
                </head>
                <body>
                  <h1>¡Hola, Mundo!</h1>
                </body>
              </html>

Este HTML se representa en el DOM como una estructura de nodos:

            
              Document
              ├─ html
              │  ├─ head
              │  │  └─ title
              │  │     └─ #text (Mi Página)
              │  └─ body
              │     └─ h1
              │        └─ #text (¡Hola, Mundo!)

Etiquetas HTML

En HTML, las etiquetas se utilizan para marcar partes específicas del contenido y darle formato. Cada etiqueta tiene una sintaxis específica.

Sintaxis de Etiqueta HTML:

<etiqueta>Contenido</etiqueta>

Ejemplos de Etiquetas Comunes:

  • <p> - Párrafo
  • <a> - Enlace
  • <img> - Imagen
  • <ul>, <ol>, <li> - Listas
  • <h1>, <h2>, ..., <h6> - Encabezados
  • <div> - Contenedor genérico

Ejemplo de Etiquetas HTML:

            
              <body>
                <h1>Mi Página</h1>
                <p>¡Bienvenido a mi página web!</p>
                <img src="imagen.jpg" alt="Descripción de la imagen">
                <a href="https://www.ejemplo.com">Enlace de Ejemplo</a>
              </body>

Mi Aprendizaje de HTML y CSS

La estructura básica de un proyecto HTML incluye etiquetas típicas que definen la cabeza, el cuerpo y otros elementos esenciales.

Etiquetas Típicas:

  • <!DOCTYPE html> - Declaración del tipo de documento
  • <html> - Elemento raíz
  • <head> - Encabezado del documento
  • <title> - Título del documento
  • <body> - Cuerpo del documento

Estructura Básica del Proyecto:

            
              <!DOCTYPE html>
              <html lang="es">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <link rel="stylesheet" href="styles.css">
                <title>Mi Proyecto HTML</title>
              </head>
              <body>
                <header>
                  <h1>Bienvenido a Mi Proyecto HTML</h1>
                </header>
                <main>
                  <p>Este es un proyecto de ejemplo</p>
                </main>
                <footer>
                  <p>© 2023 Mi Proyecto HTML</p>
                </footer>
              </body>
              </html>

Atributos HTML

Los atributos se utilizan para proporcionar información adicional sobre las etiquetas HTML. Cada atributo tiene un propósito específico y se utiliza con etiquetas particulares.

Ejemplos según Etiqueta:

  • <a> - href, target
  • <img> - src, alt
  • <input> - type, name, value
  • <div> - class, id

Ejemplo con Atributos:

            
              <a href="https://www.ejemplo.com" target="_blank">Enlace Externo</a>
              <img src="imagen.jpg" alt="Descripción de la Imagen">
              <input type="text" name="username" value="Usuario">
              <div class="contenedor" id="miDiv">Contenido del Div</div>
